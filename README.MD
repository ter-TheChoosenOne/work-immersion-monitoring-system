## Description
A comprehensive work immersion system providing a portal for teachers and students to manage immersion activities, attendance, reports, and administrative functions.

## Features
- User authentication and authorization
- Dashboard for students and teachers
- Admin panel for system management
- Attendance tracking and reporting
- User management (students, teachers, admins)
- System configuration and backup
- Security settings and access logs

## Technology Stack
- **Frontend**: React, React Router, Axios, Socket.io-client, Recharts
- **Backend**: Node.js (assuming from context)
- **Testing**: Vitest, Jest, React Testing Library

## Installation
1. Clone the repository
2. Install dependencies for frontend and backend
3. Configure environment variables
4. Run the development servers

## Usage

### Getting Started
1. Start the backend server
2. Start the frontend development server
3. Access the application at the provided live links above

### How to Use the Website

#### Step-by-Step Guide

1. **Access the Application**
   - Navigate to the [Frontend Web Application](https://work-immersion-monitoring-system-po.vercel.app) [Admin -](https://work-immersion-monitoring-system-po.vercel.app/admin/login)  in your web browser
   - You'll see the splash screen loading the Work Immersion System

2. **Login Options**
   The system supports three types of users: Admin, Teacher, and Student

3. **Admin Login**
   - Click on "Admin Login" or navigate to `/admin/login`
   - Use the following credentials:
     - **Email**: hero@example.com
     - **Password**: adminpassword1234

4. **Teacher Login**
   - Click on "Login" or navigate to `/login`
   - Select teacher login type if prompted
   - Use the following sample credentials:
     - **Teacher ID**: 23B123
     - **Password**: Example#1

5. **Student Login**
   - Click on "Login" or navigate to `/login`
   - Select student login type if prompted
   - Use the following sample credentials:
     - **Student ID**: 23B6767
     - **Password**: Test12345

6. **Exploring the Dashboard**
   - After successful login, you'll be redirected to your respective dashboard
   - Admin users can access admin-specific features like user management, system configuration, etc.
   - Teachers and students can view their respective dashboards with relevant information

7. **Navigation**
   - Use the navigation menu to access different sections
   - Admin users have access to all administrative features
   - Teachers can manage their classes and view reports
   - Students can view their immersion activities and attendance

### Additional Features
- **Forgot Password**: Use the "Forgot Password" link if you need to reset your password
- **Register**: New users can register through the registration page
- **Security**: All passwords are securely handled and stored
